# 需求文件 (Requirements)

## 1. 專案目標
為**音樂創作者**打造一款 iPhone 原生鋼琴 App。傳統琴鍵介面在手機上難以操作，本 App 改採**手遊式虛擬類比按鍵**，讓使用者能單手或雙手快速彈奏旋律與和弦，並支援**滑動推弦**與**力度動態**等表現力。

## 2. 目標使用者
- 想隨時記錄靈感的作曲者 / 編曲者
- 鋼琴初學者 / 業餘彈奏者
- 對「手機上彈不好鋼琴」感到挫折的人

## 3. 核心功能需求

### 3.1 虛擬類比按鍵 (Analog Pad)
- **預設配置**：10 顆圓形按鍵，對應 C 大調 1 個八度 + 上下各 1 音
  `低音 Si – Do – Re – Mi – Fa – Sol – La – Si – 高音 Do – 高音 Re`
- **多點觸控**：至少同時支援 5 指按壓 (彈奏和弦)
- **雙重類比輸入**：
  1. **力度 (Velocity)** — 採**雙來源混合**偵測，對應 MIDI velocity 1–127：
     - **A. 加速度計 (CMMotionManager)** — 手指敲擊瞬間的裝置震動量 (主要)
     - **B. 觸碰半徑 (majorRadius)** — 手指接觸面積 (後備 / 平放桌上時)
     - 系統自動偵測使用情境（手持 vs 平放）切換主來源
     - 提供**校準流程**：使用者輕敲 3 次、重敲 3 次，建立個人化對應曲線
  2. **滑動推弦 (Pitch Bend)** — 在按鍵上向上/向下/左右滑動可推半音 (±1 semitone)，模擬吉他推弦
- **視覺回饋**：按下時按鍵發光、漣漪動畫，滑動時顯示推弦量

### 3.2 觸覺回饋 (Haptic Feedback)
- 按下瞬間震動 (預設 0.05–0.3 秒，可調)
- 震動**強度**與**時長**可由使用者調整
- 推弦時可選擇是否有持續性微震動
- 使用 **Core Haptics** (CHHapticEngine) 支援細緻的力度對應

### 3.3 自訂功能 (Customization)
- 按鍵的**新增 / 刪除 / 拖曳重排**
- 每顆按鍵可指定**音高** (MIDI note)
- 按鍵**大小**與**位置**可自由調整 (編輯模式)
- **預設組合**儲存與切換 (例如「C 大調」、「A 小調」、「五聲音階」)
- 觸覺回饋強度 / 時長設定
- 滑動推弦的**靈敏度**與**最大半音範圍** (預設 ±1，可開放到 ±2)
- 力度感應**模式切換** (加速度計 / 半徑 / 兩者混合) 與**校準**

### 3.4 音色 (Sound)
- 採用**可商用的開源鋼琴音色**
  - 候選：**Salamander Grand Piano V3** (CC BY 3.0) — 高品質演奏級取樣
  - 備案：**FluidR3_GM** (MIT-like) — 體積較小的 GM SoundFont
- 音色檔案打包在 App 內，離線可用
- 支援多重採樣 (依 velocity 切換取樣層)

### 3.5 創作工具 (Creator Tools)

#### Phase 4a — 單軌錄音
- **錄音**：可錄下彈奏，輸出為 `.m4a` (audio) 或 `.mid` (MIDI)
- **節拍器**：可調 BPM、拍號
- **回放**：錄完後可立即試聽
- **匯出 / 分享**：透過 iOS 標準分享面板輸出檔案

#### Phase 4b — 多軌與樂譜
- **多軌錄音**：至少支援 **2 軌** (旋律 / Bassline)
  - 可分別錄製：先錄旋律，邊聽邊錄 bass，反之亦然
  - 每軌可獨立**靜音 / Solo / 刪除 / 重錄**
- **樂譜輸出**：
  - **MusicXML** (`.musicxml`) — 可匯入 MuseScore / Finale / Sibelius / Logic Pro
  - **PDF 樂譜** — 高音譜表 + 低音譜表雙 staff，可列印或預覽
  - 透過 **Verovio** (LGPL) 渲染，自製 `MusicXMLWriter` 組裝資料
- **多軌播放**：兩軌同步播放，可調各軌音量

## 4. 非功能需求

### 4.1 效能
- 觸碰到出聲延遲 **< 20 ms** (使用 AVAudioEngine + 低 buffer size)
- 60 FPS UI，動畫不卡頓

### 4.2 平台
- iOS 16.0+ (使用較新的 SwiftUI / Core Haptics API)
- iPhone **僅支援橫向** (`landscapeLeft` + `landscapeRight`)，鎖定不旋轉直立
- 暫不支援 iPad / Mac (未來可擴充)

### 4.3 技術
- **原生 Swift + SwiftUI** (UI)
- **AVAudioEngine + AVAudioUnitSampler** (音訊引擎)
- **Core Haptics** (觸覺回饋)
- **CMMotionManager** (加速度計 — 力度偵測)
- **Verovio** (樂譜渲染) + **PDFKit** (PDF 輸出)
- 無後端，純單機 App

### 4.4 法律
- 所有取樣音色必須**可商用**且授權清楚 (CC BY / MIT / Public Domain)
- App 內標示音色來源與授權

## 5. 不在本期範圍 (Out of Scope)
- 多人協作 / 雲端同步
- AI 自動伴奏 / 自動和弦辨識
- 3 軌以上的多軌錄音 (本期僅 2 軌)
- 樂譜的**手動編輯** (僅輸出，不在 App 內編輯音符)
- iPad 專屬版面
- 第三方 MIDI 控制器外接

## 6. 成功標準
- 使用者可在 30 秒內彈出一段簡單旋律
- 推弦感受接近真實「按壓 + 滑動」的直覺
- 力度有明顯層次（pp / mf / ff 可清楚區分）
- 創作者錄製的 MIDI / MusicXML 可匯入 GarageBand / MuseScore 正常播放
- 可錄旋律 + bassline 兩軌並輸出可閱讀的 PDF 樂譜
